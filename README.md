# Student-Teacher Appointment Booking System

This is a web-based appointment booking system built using HTML, CSS, JavaScript and Firebase.

## Features
- Student Registration & Login
- Teacher Registration
- Admin Panel
- Teacher Dashboard
- Student Dashboard
- Book Appointments
- Approve / Cancel Appointments
- Firebase Database Integration

## Technologies
- HTML
- CSS
- JavaScript
- Firebase (Firestore)

## How to Run
1. Create a Firebase project
2. Enable Firestore
3. Paste Firebase config inside app.js
4. Open index.html in browser
5. Register and login

## Modules
- Admin: Manage teachers and students
- Teacher: Approve or cancel appointments
- Student: Search teacher and book appointments
